## Caveat

The `<DateFieldPrimitive />` provides a basic date input functionality, but it lacks accessibility features found in native date inputs or comprehensive libraries like [React Aria Components](https://react-spectrum.adobe.com/react-aria/components.html)'s [DateField](https://react-spectrum.adobe.com/react-aria/DateField.html). For instance, it does not support segmented input fields for day, month, and year, nor does it allow for keyboard-based adjustments (e.g., using arrow keys to increment or decrement dates).

While building a fully accessible date field from scratch presents significant challenges, this component meets the current needs of our DatePicker. Given the lack of modular alternatives with similar accessibility features, this implementation provides a lightweight, suitable solution for now.
